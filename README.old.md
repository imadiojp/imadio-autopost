# imadio autopost

X（旧Twitter）の自動投稿に特化した無料ツールのハイファイモックアップです。

## 特徴

- **スレ自動分割**: 長文を自動的に280文字ずつ分割してスレッド投稿
- **画像・Alt対応**: 最大4枚の画像とAltテキストを設定可能
- **ドラッグ移動**: カレンダー上で投稿をドラッグして時間変更（開発中）
- **一括停止**: すべての予約投稿を一時的に停止
- **自動リトライ**: 失敗した投稿を自動で最大3回まで再試行
- **投稿履歴**: 過去の投稿を一覧で確認、フィルタリング可能
- **メール通知**: 投稿の成功・失敗をメールで通知

## 技術スタック

- **React** + **TypeScript**
- **Tailwind CSS** (カスタムデザイントークン)
- **Shadcn/ui** (UIコンポーネントライブラリ)
- **Lucide React** (アイコン)
- **Sonner** (トースト通知)
- **Vite** (ビルドツール)

## デザイン原則

- 余白を多く取る
- 8ptグリッドシステムを徹底
- 日本語UIに最適化
- ライト/ダークテーマ完全対応
- Noto Sans JP + Interフォント
- **完全レスポンシブ対応**（モバイル、タブレット、デスクトップ）

## レスポンシブ対応

### ブレークポイント
- **モバイル**: 〜767px（ハンバーガーメニュー、シングルカラム）
- **タブレット**: 768px〜1023px（一部2カラム）
- **デスクトップ**: 1024px〜（サイドバー常時表示、フルレイアウト）

### 主な対応
- サイドバーがモバイルでオーバーレイメニューに自動変換
- テーブルとカレンダーに横スクロール対応
- フォームフィールドがモバイルで縦積み
- ボタンとタッチターゲットが44px以上を確保
- カスタムスクロールバーでUX向上

## 開発

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:5173](http://localhost:5173) を開いてください。

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## ページ一覧

1. **投稿作成 (Composer)** - テキスト、画像、予約日時の設定
2. **カレンダー (CalendarView)** - 週表示カレンダーで予約投稿を管理
3. **予約キュー (Queue)** - 予約投稿一覧と一括操作
4. **履歴 (History)** - 過去の投稿履歴をテーブル表示
5. **設定 (Settings)** - タイムゾーン、一括停止、メール通知、自動リトライ
6. **X連携 (Connections)** - Xアカウントの連携管理
7. **Empty States** - 空状態パターンのデモ
8. **マーケティング** - プロダクト紹介ページ
